# Supermarket Sales Intelligence Dashboard

This project is an interactive Tableau dashboard developed to analyse supermarket sales performance, customer behaviour, product category trends, and revenue insights.

## Tools Used
- Tableau
- SQL
- Excel / CSV

## Features
- Sales trend analysis
- KPI dashboard
- Customer behaviour analysis
- Product category performance tracking
- Revenue visualisation

## Project Goal
To transform raw sales data into actionable business insights using data visualisation techniques.
